发表于 2023-03-01 更新于 2023-03-03

跟着教程学习搭建一个博客

学习的教程地址在本文末尾

首先让我们来安装git安装nodejs安装npm以及安装之后用来连接云服务器的finalshell吧。

安装完git后通过右键直接打开git bash，且git自带ssh（Secure Shell），通过输入ssh可以查看。

$ ssh
usage: ssh [-46AaCfGgKkMNnqsTtVvXxYy] [-B bind_interface]
       [-b bind_address] [-c cipher_spec] [-D [bind_address:]port]
       [-E log_file] [-e escape_char] [-F configfile] [-I pkcs11]
       [-i identity_file] [-J [user@]host[:port]] [-L address]
       [-l login_name] [-m mac_spec] [-O ctl_cmd] [-o option] [-p port]
       [-Q query_option] [-R address] [-S ctl_path] [-W host:port]
       [-w local_tun[:remote_tun]] destination [command [argument ...]]

输入这个命令采用RSA算法生成密钥和公钥，生成的文件在users/用户/.ssh 目录下，使用记事本打开文件id_rsa.pub，并复制内容添加到github网站中的Settings->SSH and GPG keys->new SSH key
使用公钥授权的方式，在使用git上传文件时就不需要每次登录输入口令了。

ssh-keygen -t rsa

通过在git bash中输入指令

ssh -T git@github.com

进行检验，如果警告是否确定继续连接，只需输入yes同意。

提交文件：
在github中创建repository，并复制克隆地址，在git bash中clone下来，得到远程仓库。

Oh shxt，我忘记把目录切回git codeline了，所以我把这个仓库clone到.ssh目录下了。

这样才对。

现在在这个仓库下添加一个test.txt的文本文件，再在git bash中进入这个目录下，输入git status命令查看仓库状态。

输入

git add test.txt
git commit -m "commit test file"

第一次提交还需要输入用户名和邮箱，在git bash中会有命令提示
等用户名和邮箱设置好后再执行

git commit -m "commit test file"

然后输入命令 git push origin main即可。

再在github仓库上刷新就可以看到刚刚提交的test更新了。

接下来整个博客搭建最有含金量的地方来了：

价值9块的域名购买。

但是要先实名认证购买时需要审核信息，趁这个时候让我们在cdm中使用脚手架安装一下webpack吧。
注意，安装webpack之前确保安装了node.js，可通过node -v和npm -v来查看安装版本。
以及添加环境变量和路径。

oh shxt！卡住了！
使用ctrl c强制退出，设置一下国内（淘宝，如下）代理就能光速下载了

 npm config set registry https://registry.npm.taobao.org

现在再在github中创建一个仓库，名为: 用户名.github.io

再接着安装Hexo框架。
使用npm install -g hexo-cli安装，安装完后使用hexo init初始化。
但是又出错了，显示找不到hexo命令。（这种情况还是比较容易知道出错的原因的，说明需要在环境变量中添加hexo-cli/bin路径）
添加完后重新启动git bash再输入hexo init初始化即可。
（其实这里又出了一些错，原因是对某些文件夹的访问权限不够，只需要找到git bash快捷方式，将其属性设置为以管理员方式运行）

之后输入hexo g部署 hexo s查看状态
接下来在浏览器输入localhost:4000

Holy！终于！

接下来需要将hexo部署至GitHub
在blog文件夹中找到_config.yml文件在最底部添加： 

deploy:
  type: git
  repository: https://github.com/Psermoah/Psermoah.github.io.git 
  branch: main

接着安装git部署插件

npm install hexo-deployer-git --save

输入

hexo clean
hexo g
hexo d

现在在浏览器输入：https://Psermoah.github.io


我直接狠狠访问！

接下来基本没什么事了。
安装主题：
在_config.yml中配置一下标题描述时区，还有语言主题之类。
然后再一次hexo clean & hexo g & hexo d即可

就像这样。
不过还有一些需要注意的点。
使用hexo s 可以在本地实时预览页面效果，但是真正在部署时一套hexo clean & hexo g & hexo d三连下来立刻去访问可能页面不会发生改变，因为有延迟。
